A macro for including the source file and line number where a log occurred.

Parameters
formatA format string, which must not be nil.
...A comma-separated list of arguments to substitute into format.

This is defined as a compiler macro so it can automatically fill in the file name and line number where the call was made. After printing these values in brackets, this macro calls CHQuietLog with format and any other arguments supplied afterward.

OBJC_EXPORT void CHQuietLog ( NSString *  format,
  ... 
)

Provides a more terse alternative to NSLog() which accepts the same parameters.

The output is made shorter by excluding the date stamp and process information which NSLog prints before the actual specified output.

Parameters
formatA format string, which must not be nil.
...A comma-separated list of arguments to substitute into format.

Read Formatting String Objects and String Format Specifiers on this webpage for details about using format strings. Look for examples that use NSLog() since the parameters and syntax are idential.
